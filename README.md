# practice-community-front :: 실천공동체

## 프로젝트 주제
- 게시판 및 대시보드 생성 프로젝트 프론트엔드 코드입니다.
- 모든 게시판을 통합 관리시스템과 각각의 게시판 별로 확장성있게 기능들을 탑재할 수 있도록 구현하고자 합니다.
